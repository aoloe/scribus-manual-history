1.4 Fragen und Antworten zur Installation



1.4.1 Scribus startet nicht: Es werden keine Fonts gefunden

Antworten von Christian Töpp:
  • Stellen Sie sicher, dass /usr/X11/lib/X11/fonts/Type1 oder ein ähnlicher Pfad existiert
  • Wenn dieser Pfad existiert aber nicht in der Ausgabe des Programms "xset -q" im Bereich Type 1 Fonts auftaucht, ergänzen Sie Ihre XF86Config um diesen Pfad
  • Stellen Sie sicher, dass die XF86Config in der Sektion Modules einen Eintrag "Load Type1" enthält
  • Starten Sie X neu
  • Überprüfen Sie erneut, ob "xset -q" nun die Pfadangabe zu Ihren Type1-Fonts enthält
