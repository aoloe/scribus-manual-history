2.1 Die Menüs



2.1.1 Das Menü File
New Erstellt ein neues Dokument
Open Öffnet ein Dokument
Recent Documents Zeigt die zuletzt bearbeiteten Dokumente an
Close Schließt ein geöffnetes Dokument
Save Speichert das aktuelle Dokument
Save as Speichert das aktuelle Dokument unter einem anderen Namen
Get Text/Picture Lädt einen Text bzw. ein Bild in den aktuellen Rahmen
Save Text Speichert den Text eines Rahmens in einer Textdatei
Save Page as EPS Erstellt aus der aktuellen Seite eine EPS-Datei
Save as PDF Speichert das aktuelle Dokuemnt als PDF-Datei
Document Info Enthält Beschreibungen zu dem aktuellen Dokument
Document Setup Einstellungen zu Seitengrößen etc.
Print Druckt das Dokument
Quit Beendet Scribus


2.1.2 Edit
Cut
Copy Kopieren
Paste Einfügen
Clear Löschen
Select All Alles auswählen
Colors Farbauswahl
Styles Einstellungen zu Schriftart etc.
Templates Dokument-Vorlagen
Fonts Schriftartenauswahl
Preferences Allgemeine Scribus-Einstellungen


2.1.3 Style
Bei ausgewähltem Text bzw. Textrahmen:
Font Zur Formatierung verfügbare Schriftarten
Size Schriftgröße
Style Text-Auszeichnungen wie z.B. Fett, kursiv etc.
Color Schrfitfarbe
Shade Abtönung

Bei ausgewähltem Bildrahmen:
Color Rahmenfarbe
Invert Negativ


2.1.4 Item
Modify Modifiziert das aktuelle Objekt
Duplicate Dupliziert das aktuelle Objekt
Multiple Duplicate Dupliziert das aktuelle Objekt mehrfach
Delete Löscht das aktuelle Objekt
Send to back Stellt das aktuelle Objekt ganz in den Hintergrund
Bring to front Stellt das aktuelle Objekt ganz in den Vordergrund
Lower Stellt das akktuelle Objekt eine Ebene weiter nach hinten
Raise Stellt das aktuelle Objekt eine Ebene weiter nach vorne
Distribute again ???
Shape Rahmenform des aktuellen Objekts


2.1.5 Page
Insert Fügt dem Dokument eine neue Seite hinzu
Delete Löscht eine Seite des Dokuments
Move Verschiebt eine Seite des Dokuments
Apply Template Wendet eine Dokumentvorlage auf das aktuelle Dokument an

Thumbnails Zeigt alle Seiten des Dokuments als kleine Vorschaubilder an

Hide / Show Frames Blendet Rahmenkanten ein / aus
Hide / Show Images Blendet Bilder aus / ein
Show / Hide Guides Blendet ein Hilfsliniengitter ein / aus
Snap to Guides Richtet Objekte am Hilfsliniengitter aus
Show / Hide Tools Blendet die Werkzeugleiste ein / aus
