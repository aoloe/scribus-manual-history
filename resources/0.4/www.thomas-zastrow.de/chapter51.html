5.1 Generelles

In diesem Tutorial soll anhand eines Buch-Covers die Bedienung von Scribus erklärt werden.
Alle Dateien, die zum Nachvollziehen dieses Tutorials nötig sind, finden Sie im Unterordner "tutorial". Die Datei "book_cover.doc" enthält das eigentliche Dokument, "book_cover.pdf" einen PDF-Ausdruck der Datei.
Als einzige Schriftart wurde "Charter Bold" benutzt. Unter Umständen ist diese Schriftart nicht auf Ihrem System vorhanden, Sie können aber auch jede beliebige andere oder auch verschiedene verwenden.
