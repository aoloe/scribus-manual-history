1.2 Download und Installation

Sie können die neueste Version von Scribus jeweils unter

http://web2.altmuehlnet.de/fschmid/

downloaden. Entpacken Sie das Archiv mittels

tar zxvf scribus-x.x.x.tar.gz

Wechseln Sie in das neu erstellte Verzeichnis und geben Sie nacheinander

configure
make
make install (hierfür benötigen Sie Root-Rechte)
make clean

ein um das Programm zu kompilieren. Da Scribus autoconf benutzt sollten Sie keine Probleme beim Kompilieren haben. Falls doch, melden Sie Ihre Probleme dem Entwickler unter Franz.Schmid@altmuehlnet.de
