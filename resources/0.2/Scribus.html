Scribus

What is Scribus?

Scribus is a Layoutprogram for Linux, similar to Pagemaker, Xpress or InDesign, except that it is published under the GPL.

For a quick Overview how Scribus looks like see: here

Unlike other Programs Scribus uses only Type1-Fonts of the X-Server. Therefore there is no fiddling around with installing extra Fonts. For this reason the number of Fonts is a little bit limited, but you can be sure that your Monitor shows exacty the same as the printed Output is

What do you need for Scribus?

To use Scribus you must have installed:

  • QT Version 2.2.2
  • Python Version 2.0
  • PyQt Version 2.2
  • Ghostscript Version 5.50
  • ImageMagick
  • pfb2pfa
  • Postscript (Type1) Fonts

For PDF-Export required:

  • PDFlib Version 3.03

Installation:

  1. Unpack the File Scribus.tar.gz with tar xfz Scribus.tar.gz
  2. Copy the Folder Scribus to /usr/local
  3. Create a Symlink from /usr/local/Scribus/Scribus.py to /usr/local/bin/Scribus
  4. If you wish to create PDF-Documents adjust the File pdflib.upr in the Scribus-Folder to match your System
